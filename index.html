Moje Projekty

Shop Electro

Pełnoprawna aplikacja e-commerce do sprzedaży elektroniki. Posiada system rejestracji i logowania użytkowników, panel administratora do zarządzania produktami (dodawanie, edycja, usuwanie), koszyk zakupowy oraz wbudowany czat w czasie rzeczywistym między klientem a administratorem. Baza danych przechowuje produkty, użytkowników i historię zamówień.

  • Logowanie / Rejestracja
  • Panel Admina
  • Koszyk
  • Czat live
  • CRUD produktów
HTML • CSS • JS • PHP • SQL

Mini Task Manager

Prosta i przejrzysta aplikacja do zarządzania codziennymi zadaniami. Pozwala tworzyć, oznaczać jako ukończone i usuwać zadania — wszystko bez przeładowania strony. Dane są przechowywane w localStorage, dzięki czemu zadania nie znikają po zamknięciu przeglądarki. Interfejs jest responsywny i działa na urządzeniach mobilnych.

  • Dodawanie zadań
  • Oznaczanie jako done
  • localStorage
  • Responsywny UI
JavaScript • HTML • CSS

My Cinema Site

Interaktywna strona internetowa dla kina „CineFlex". Użytkownik może przeglądać repertuar, wybierać miejsca na sali kinowej w trybie wizualnym (mapa siedzeń), zamawiać przekąski z menu bufetowego oraz sprawdzać cennik biletów. Projekt zbudowany w React z dbałością o wygląd i animacje — klimatyczny dark-mode z czerwonymi akcentami.

  • Mapa siedzeń
  • Repertuar filmów
  • Menu bufetu
  • Cennik biletów
  • Dark mode UI
React • JavaScript • CSS

Weather App (API)

Aplikacja pogodowa pobierająca dane w czasie rzeczywistym z OpenWeatherMap API. Wyświetla aktualną temperaturę, odczuwalną, stan powietrza (AQI), godzinowy prognozę na 24h oraz zachód / wschód słońca dla dowolnego miasta świata. Obsługuje przełączanie jednostek °C / °F i dostępna jest prognoza na 5 dni. Całość utrzymana w eleganckim dark-mode.

  • REST API (OpenWeather)
  • Prognoza godzinowa
  • 5-dniowa prognoza
  • AQI powietrza
  • °C / °F
  • Wyszukiwanie miast
JS • REST API • JSON
